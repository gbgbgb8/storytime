How to Use MakeJSON

      

      

        

          
File and Folder Structure

          
Make sure your main project has the following structure:

          
- main_folder/
  - stories/
    - your_story_folder/
      - story.json
      - page1-a.jpg
      - page1-b.jpg
      - page2-a.jpg
      
  - makejson.html
  - makejson.js
  - makejson.css
  - script.js
  - style.css
  - index.html
          

          
Creating a New Story

          
    
            
  1. Open makejson.html in a browser.
    2. 
            
  2. Use the Add Page button to create a new page.
    3. 
            
  3. Fill in the narrative text, setting, time, and options.
    4. 
            
  4. Click Generate JSON to see the JSON output.
    5. 
            
  5. Use the Copy JSON button to copy the JSON data.
    6. 
            
  6. Create a new folder under /stories and name it according to your story.
    7. 
            
  7. In this folder, create a file named story.json and paste the copied JSON data.
    8. 
            
  8. Add images to this folder, following the naming conventions like page1-a.jpg, page1-b.jpg, etc.
    9. 
            
  9. Finally, update the script.js file to include your new story in the story dropdown.
