How to Use MakeJSON

To create a new story for the main StoryTime app, follow these steps:

  1. Click the 'Add Page' button to add a new page to your story.
  2. Fill out the Narrative Text, Setting, Time, Option A Text, Option B Text, Option A Next Page, and Option B Next Page for each page.
  3. Click the 'Generate JSON' button to generate the JSON structure of your story.
  4. Click the 'Copy JSON' button to copy this JSON to your clipboard.
  5. Paste this JSON into a new file and save it as 'story.json' inside a folder under the 'stories' directory of your StoryTime app.
